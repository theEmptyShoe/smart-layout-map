# Smart Layout Map

An interactive web map for Royal Park Residency, Bengaluru.
Built to help residents, visitors, and delivery personnel navigate the area,
and to document existing green cover as a baseline record.

**Live site:** https://theEmptyShoe.github.io/smart-layout-map

## Tech
Leaflet.js · Turf.js · HTML/CSS/JS · Python (data processing) · GeoJSON
